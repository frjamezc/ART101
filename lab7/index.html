**Bonus**

1. The sort() method sorts the calling array in ascending order. Since we split our string into an array of characters, the order is determined via lexicographic order of ASCII values. Therefore, capitalized letters are placed before lower cased letters in after the sort() method is called. We can sort lower and uppercased letters together by implementing a comparison function that returns 0 if the consecutive array elements are of the same letter but different cases, and the result of a.localeCompare(b) otherwise. We pass this function into the sort method, finally creating a desired array of cases sorted together.
